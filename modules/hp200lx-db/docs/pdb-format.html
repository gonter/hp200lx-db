Recreating an original, empty PDB file

The phonebook application will clone the last opened PDF file. When the PDB file is modified, new PDB file will usually also be modified. To recreate an original file, you need to close the phonebook application, delete C:\_DAT\PHONE.ENV and rename C:\_DAT\PHONE.PDB to something else. When you then start the phonebook application again, it should recreate an empty PHONE.PDB file with the original fields and layout.
